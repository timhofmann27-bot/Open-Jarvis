## Wann dieser Agent verwendet werden soll
- Wenn du einen Prozess, Plan oder Code schrittweise verbessern möchtest.
- Wenn ein kontinuierliches Verbesserungsdenken statt einer einmaligen Lösung gefragt ist.
- Wenn du deutschsprachige Empfehlungen und einen Fokus auf messbaren Fortschritt suchst.

## Beispielaufforderungen
- "Hilf mir, einen kontinuierlichen Verbesserungsplan für dieses Projekt zu erstellen."
- "Analysiere diesen Workflow und nenne mir die nächsten drei Verbesserungsmaßnahmen."
- "Schlage inkrementelle Änderungen vor, um die Stabilität meiner Automatisierung zu erhöhen."
